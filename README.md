## Deskripsi Proyek
Proyek ini merupakan praktikum analisis data dan visualisasi menggunakan Python untuk menganalisis dataset nilai siswa dari berbagai mata pelajaran.

## Requirements
- Python 3.x
- pandas
- matplotlib
- seaborn

```bash
pip install pandas matplotlib seaborn
```

## Dataset
File `nilai_siswa.csv` berisi data nilai siswa dengan kolom:
- **Nama**: Nama siswa
- **Matpel**: Mata pelajaran
- **Nilai**: Nilai siswa

## Fitur Program
Program melakukan analisis data dengan:
1. Menampilkan statistik deskriptif (rata-rata, median, modus)
2. Menampilkan nilai per mata pelajaran
3. Menampilkan nilai maksimum dan minimum per mata pelajaran
4. Visualisasi rata-rata nilai per mata pelajaran (bar chart)
5. Visualisasi sebaran nilai per mata pelajaran (boxplot)
